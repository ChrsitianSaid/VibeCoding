# VibeCoding


Crea una página web educativa y visualmente atractiva sobre la física cuántica usando HTML, CSS y JavaScript. La página debe incluir:

1. Una portada con un título llamativo y un breve texto introductorio.
2. Secciones explicando conceptos clave como: superposición, entrelazamiento, dualidad onda‑partícula y el principio de incertidumbre.
3. Animaciones o interacciones simples en JavaScript que ayuden a visualizar fenómenos cuánticos (por ejemplo, partículas moviéndose aleatoriamente, simulación básica de superposición, etc.).
4. Un diseño moderno, limpio y responsive usando CSS.
5. Una paleta de colores oscuros con acentos neón (azul, morado o verde).
6. Un menú de navegación fijo.
7. Un footer con créditos y enlaces adicionales.
8. Código organizado en archivos separados: index.html, styles.css y script.js.
9. Comentarios en el código explicando cada parte para que sea fácil de entender por estudiantes.

Genera el código completo y funcional.
